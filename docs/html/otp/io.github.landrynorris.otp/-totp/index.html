Totp

data class Totp(val secret: ByteArray, val name: String, timeStep: Int = 30, val codeLength: Int = 6) : Otp

Implementation of RFC 6238 Time-Based One-Time Password. Uses current time step as the challenge. The time step is calculated as the Unix timestamp divided by timeStep. The timestamp can be overridden using setTime.
